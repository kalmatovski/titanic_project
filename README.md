# Titanic Survival Prediction

This project predicts whether a passenger survived the Titanic disaster using machine learning.

## Project Overview

The goal of this project is to build a clean and structured ML pipeline using the Titanic dataset.

This is part of my ML learning journey, focusing on:

- data preprocessing
- model training
- evaluation
- project structure

## Features Used

- Pclass
- Sex
- Age
- Fare
- Embarked

---

## Model

- Logistic Regression

## Current Results

- Accuracy: ~0.80
